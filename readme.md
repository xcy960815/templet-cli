### templet-cli

> 一个简单的模板 cli

### 安装

```shell
 npm install templet-cli -g
```

#### 查看模版列表

```shell
tem list
```

#### 初始化指令

```shell
tem init
```

#### 创建指定模版指令

```shell
tem create <templateName> <projectName>

    templateName --> 模板名称  不知道模板名称可以执行 tem list 查看

    projectName --> 项目名称   自己的项目名称
```

#### 更新指令

```shell
tem update

```

#### 帮助指令

```shell
tem help
```
